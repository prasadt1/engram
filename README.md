# Engram

**An AI photography coach that remembers you — and knows what to forget.**

Engram is a forgetting-aware memory engine for Qwen agents (salience-scored recall, progress-driven forgetting, episodic-to-semantic consolidation, token-budget context packing), shipped as a custom `engram-mcp` server — with a longitudinal photography coach as its flagship application.

Built for the Global AI Hackathon with Qwen Cloud (Track 1: MemoryAgent).

Built on [Iris](https://github.com/prasadt1/iris-photography-mentor), my open-source (Apache-2.0) photography mentor foundation. Everything memory-engine, Qwen, and Alibaba Cloud here is new work from the submission period — see WHATS_NEW.md.
